# miniguia-estudos-notebooklm
um miniguia de estudo com resumos estruturados, glossário de conceitos e um conjunto de prompts reutilizáveis que apoiem futuras revisões. A atividade enfatiza o uso da IA como ferramenta de aprendizagem ativa, aliando pensamento crítico, curadoria de fontes e organização do conhecimento.

📝 DESCRIÇÃO DO TEMA ESTUDADO
O tema estudado neste projeto é uma introdução prática ao universo de dados, voltada para pessoas que estão dando seus primeiros passos na área. A proposta foi utilizar o NotebookLM como ferramenta de aprendizagem ativa, reunindo diferentes fontes em texto e vídeo para compreender conceitos fundamentais como tipos de dados, manipulação com Python e Pandas, visualização básica e noções iniciais de SQL.

A partir desse material, foram definidos objetivos de estudo, elaboradas perguntas estratégicas e testadas variações de prompts para explorar como a IA pode apoiar o processo de aprendizado. O resultado é um miniguia estruturado que reúne resumos, glossário de conceitos essenciais e um conjunto de prompts reutilizáveis, permitindo revisões futuras e aprofundamento contínuo.

Este projeto demonstra como a Inteligência Artificial pode ser integrada ao estudo de forma crítica e organizada, fortalecendo habilidades de curadoria, síntese e raciocínio analítico — competências fundamentais para quem deseja iniciar sua jornada na área de dados.

# 🎯 OBJETIVO  
O objetivo deste notebook é introduzir, de forma simples e prática, os principais conceitos necessários para quem deseja dar os primeiros passos na área de dados. Aqui, você aprenderá o que significa trabalhar com dados, quais são os tipos mais comuns, como manipulá‑los usando Python e Pandas, como criar visualizações básicas e como realizar pequenas análises exploratórias. Ao final, você terá uma base sólida para continuar sua jornada rumo à Engenharia de Dados, Ciência de Dados ou Análise de Dados.
